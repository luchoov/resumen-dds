/

### Windows

1. Ingresar a la pagina oficial de Node.js: [https://nodejs.org/es](https://nodejs.org/es) y seleccionar la versión LTS 18.15.0:

![Untitled](Proceso%20de%20Instalacion%20de%20Node%20js%20Version%20LTS%2018%201%204ca378fa92a04024bb4d2be110b1146e/Untitled.png)

1. Ejecutar el archivo .msi o ejecutable descargado desde la pagina web:

![Untitled](Proceso%20de%20Instalacion%20de%20Node%20js%20Version%20LTS%2018%201%204ca378fa92a04024bb4d2be110b1146e/Untitled%201.png)

1. Confirmar las bases y condiciones y seleccionar la ubicación deseada para Node.js:

![Untitled](Proceso%20de%20Instalacion%20de%20Node%20js%20Version%20LTS%2018%201%204ca378fa92a04024bb4d2be110b1146e/Untitled%202.png)

1. Continuar sin modificar la configuración hasta la pantalla final de instalación de Node.js: 

![Untitled](Proceso%20de%20Instalacion%20de%20Node%20js%20Version%20LTS%2018%201%204ca378fa92a04024bb4d2be110b1146e/Untitled%203.png)

1. Instalar la versión LTS 18.15.0 de Node.js:

![Untitled](Proceso%20de%20Instalacion%20de%20Node%20js%20Version%20LTS%2018%201%204ca378fa92a04024bb4d2be110b1146e/Untitled%204.png)

1. Para verificar que la versión de Node.js que se instalo es la correcta abrir una terminal y ejecutar el comando ***`node -v` :***

![Untitled](Proceso%20de%20Instalacion%20de%20Node%20js%20Version%20LTS%2018%201%204ca378fa92a04024bb4d2be110b1146e/Untitled%205.png)

---

### Para Linux

1. Abrir una terminal y ejecutar el comando `**sudo apt install node**js` para instalar el paquete de Node.js para Linux.
2. Ejecutar el comando `sudo apt install npm` para instalar el gestor de paquetes de Node.js.
3. Para verificar que la versión de Node.js que se instalo es la LTS 18.15.0 ejecutar el comando `node -v`.
