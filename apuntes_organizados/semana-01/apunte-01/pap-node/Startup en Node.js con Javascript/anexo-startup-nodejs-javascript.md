# Startup en Node.js con Javascript

Una ejercitación sencilla para inicializar en Javascript y poder correr código a través de Node.js es crear un archivo que solamente muestre un mensaje **’Hola Mundo’**. Para ello:

1. Abrimos el editor de código de preferencia, en este caso usaremos **Visual Studio Code,** y nos moveremos a un directorio donde trabajar:

![Untitled](Inicializacion%20en%20Node%20js%20con%20Javascript%20ecea2f8b35e240948fe4a0e219cf2f5b/Untitled.png)

1. Crearemos un archivo dentro de este directorio con el nombre de su preferencia, nosotros lo llamaremos ‘hello’, este archivo debe tener como extensión **.js**:

![Untitled](Inicializacion%20en%20Node%20js%20con%20Javascript%20ecea2f8b35e240948fe4a0e219cf2f5b/Untitled%201.png)

![Untitled](Inicializacion%20en%20Node%20js%20con%20Javascript%20ecea2f8b35e240948fe4a0e219cf2f5b/Untitled%202.png)

1. Ahora que tenemos nuestro archivo de Javascript escribiremos en la primera linea `console.log(’Hola Mundo’);` , esto escribirá sobre la terminal el texto **‘Hola Mundo’** una vez que se ejecute el archivo **hello.js:**

![Untitled](Inicializacion%20en%20Node%20js%20con%20Javascript%20ecea2f8b35e240948fe4a0e219cf2f5b/Untitled%203.png)

1. Una vez guardado el archivo abriremos una terminal desde Visual Studio Code para ejecutar el archivo de javascript:

![Untitled](Inicializacion%20en%20Node%20js%20con%20Javascript%20ecea2f8b35e240948fe4a0e219cf2f5b/Untitled%204.png)

![Untitled](Inicializacion%20en%20Node%20js%20con%20Javascript%20ecea2f8b35e240948fe4a0e219cf2f5b/Untitled%205.png)

1. En la terminal ejecutaremos el comando `node hello.js` sustituyendo ‘hello’ por el nombre que se le haya dado al archivo de Javascript:

![Untitled](Inicializacion%20en%20Node%20js%20con%20Javascript%20ecea2f8b35e240948fe4a0e219cf2f5b/Untitled%206.png)

De esta forma se ejecutan los archivos Javascript a través de una terminal.
