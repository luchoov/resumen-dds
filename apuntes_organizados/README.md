# Apuntes organizados

Formato uniforme:
- Apuntes principales: `semana-XX/apunte-XX/apunte-XX-<tema>.md`
- Material complementario: `anexo-*`

- `apunte-02-git-control-versiones.md`: control de versiones, Git, flujo local y remoto, ramas y herramientas graficas.

## Semana 02
- `apunte-03-desarrollo-web-html.md`: desarrollo web, HTML, estructura basica, etiquetas, atributos y formularios.
- `apunte-04-css-hojas-de-estilo.md`: CSS, reglas, selectores, cascada, variables y aplicacion de estilos.

## Semana 03
- `apunte-05-maquetacion-web-bootstrap.md`: maquetacion web, modelo de cajas y Bootstrap responsive.

## Semana 04
- `apunte-06-javascript-parte-1.md`: variables, tipos, cadenas, estructuras de control, funciones y base de JavaScript.
- `apunte-07-javascript-parte-2.md`: objetos, arrays, funciones flecha, modulos, JSON y programacion defensiva.
- `apunte-08-spa-dom.md`: DOM, SPA, rutas en el cliente, hash routing y SEO en SPAs.

## Semana 05

- `apunte-10-asincronia-fetch-async-await.md`: callbacks, promesas, Map, JSON, Fetch y async/await.

## Semana 06
- `apunte-11-sequelize-orm.md`: bases de datos relacionales, ORM, Sequelize, modelos, relaciones, transacciones y repositorios.

- `apunte-13-express.md`: Express, instalacion, request/response, params y query.

## Semana 07
- `apunte-14-express-router-cors-estaticos.md`: router de Express, archivos estaticos, path, dirname y CORS.
- `apunte-15-express-middleware-error-handling.md`: middlewares, manejo de errores, excepciones y procesos en produccion.

## Semana 08
- `apunte-16-testing-jest-vitest.md`: testing de software, TDD, Jest, Vitest y herramientas complementarias.
- `apunte-17-seguridad-web-owasp.md`: seguridad web, OWASP Top 10, API Security Top 10 y mitigaciones.
- `apunte-18-autenticacion-autorizacion-jwt-oauth.md`: autenticacion, autorizacion, hash de contrasenas, JWT, refresh tokens, cookies y OAuth.

## Semana 10
- `apunte-19-react-introduccion-vite.md`: introduccion a React, Vite, JSX, Virtual DOM, componentes y renderizado.

## Semana 11
- `apunte-21-react-router.md`: React Router, rutas anidadas, `useParams`, `useNavigate` y `useLocation`.

## Semana 12
- `apunte-23-axios-react-fetch-interceptors.md`: Axios vs Fetch, consumo de APIs, JSONPlaceholder, servicio por capas e interceptores.
